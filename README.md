# 📊 Google Play Store Analytics Dashboard

Este repositorio contiene el código fuente y la documentación de un **Dashboard Interactivo** desarrollado en Python para el análisis predictivo y exploratorio de aplicaciones de la Google Play Store. 

El proyecto fue desarrollado simulando un entorno de trabajo real bajo la metodología ágil **Scrum**, gestionando el ciclo de vida del software mediante *Issues, Ramas (Branches) y Pull Requests*.

## 🚀 Características Principales
El dashboard permite a los analistas de negocio explorar métricas críticas del ecosistema móvil mediante las siguientes funcionalidades:
* **Filtros Dinámicos:** Segmentación de datos en tiempo real por `Categoría` y `Tipo` de monetización (Gratis / De Pago).
* **Análisis de Distribución:** Gráficos de cajas (Boxplots) y barras para evaluar la saturación del mercado y la volatilidad de las calificaciones.
* **Correlaciones de Rendimiento:** Gráficos de dispersión multivariables para entender la relación entre instalaciones, reseñas y popularidad.
* **UI Optimizada:** Interfaz web construida con un *Dark Theme* para reducir la fatiga visual.
